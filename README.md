# الأول برو المحاسبي — FirstPro Accounting

تطبيق محاسبي متكامل للأندرويد مبني بـ Flutter، يدعم القيد المزدوج، تعدد العملات، إدارة المخزون بطرق تكلفة متعددة، نقاط البيع، والتقارير المالية — مع قاعدة بيانات SQLite مشفرة بالكامل (SQLCipher).

## ✨ الميزات الرئيسية

- **محاسبة قيد مزدوج** كاملة مع شجرة حسابات وفترات مالية وقيود يومية
- **تعدد العملات**: فواتير وسندات وحسابات بعملات مختلفة مع تتبع أسعار الصرف وفروقاتها
- **إدارة المخزون**: متوسط مرجح / FIFO / LIFO مع طبقات تكلفة وتسويات وجرد
- **المبيعات والمشتريات**: فواتير، مرتجعات، عروض أسعار، أوامر بيع وشراء، نقاط بيع (POS) مع ورديات
- **الكيانات**: عملاء، موردون، موظفون — بأرصدة وكشوف حساب لكل عملة
- **الصناديق والبنوك**: سندات قبض/صرف، حوالات، تسوية بنكية، صرافة عملات
- **التقارير**: ميزان مراجعة، أرباح وخسائر، تقادم الديون، كشوف PDF/Excel
- **الأمان**: تشفير SQLCipher بمفتاح في Android Keystore، قفل تطبيق ببصمة/PIN
- **عربي أولاً** مع دعم RTL كامل وواجهة إنجليزية

## 🏗️ البنية المعمارية

```
UI (شاشات + Widgets)
  → ViewModels (ChangeNotifier)
    → Repositories / Services (كل SQL هنا)
      → DatabaseHelper → SQLite مشفرة (v50+)
```

- حقن التبعيات عبر **GetIt** (`lib/core/di/service_locator.dart`)
- المبالغ المالية تُخزن **INTEGER cents** عبر `MoneyHelper` (دقة محاسبية)
- ترحيلات قاعدة بيانات مرقمة ومُختبرة (`lib/data/datasources/migrations/`)

## 🚀 البدء

```bash
flutter pub get
flutter gen-l10n        # توليد ملفات الترجمة
flutter run
```

### المتطلبات
- Flutter 3.32+ / Dart 3.8+
- Android minSdk 24
