## 2. الميزات الرئيسية

### 2.1 شجرة الحسابات
- عرض هرمي متسع للحسابات مع تسريع في التنقل
- إمكانية إنشاء حسابات بأرقام وأسماء محددة (أصول، إيرادات، مصروفات، حقوق ملكية)
- كل حساب يأخذ مكان من مجموع المحاسبة، مع أرقام مصنفة وأسماء معبرة
- **عرض كشف حساب**: رصيد أول المدة مع الحركات والرصيد النهائي لكل حساب
- **رصيد مدين/دائن**: عرض الرصيد والدائن والمدين لكل حساب
- **ترحيل الحسابات**: ترحيل تلقائي للحسابات من سنة لأخرى مع الأرصدة
- **تصدير Excel**: تصدير الحسابات لملف Excel

### 2.2 نظام السندات
- **سند مزدوج**: سند بعدة حسابات (مدين ودائن)
- **سند صرف**: سند صرف
- **سند تسوية**: سند تسوية
- **سند قبض**: سند قبض
- **سند صرف/قبض**: سند صرف وقبض
- **سند معدد**: عدة سندات معاً
- **الجرد**: قيد الجرد ونتيجة الجرد
- **مسودة**: مسودة السند ومسودة مختصرة
- دعم الصور والشرح في كل سند
- دعم العملات المتعددة في كل سند
- إمكانية الطباعة والمشاركة

### 2.3 نظام الفواتير
- فاتورة مبيعات نقدية
- فاتورة مبيعات آجلة
- فاتورة مشتريات نقدية
- فاتورة مشتريات آجلة
- مرتجع مبيعات نقدية
- مرتجع مبيعات آجلة
- مرتجع مشتريات نقدية
- مرتجع مشتريات آجلة
- أمر عرض سعر / طلب شراء / طلب بيع
- إمكانية استخراج أصناف من فاتورة سابقة
- كشف ملخص الفاتورة مع الضرائب والخصم والحسم
- طباعة فاتورة بصيغة PDF
- مشاركة الفاتورة كصورة أو PDF

### 2.4 نظام المخزون
- **ملف الأصناف**: إدارة كاملة للأصناف مع بياناتها
- **ملف المخازن**: إدارة المخازن المتعددة
- **حركات المخزون**: صادر (حالة/محررة) ووارد (حالة/محررة)
- **تحويل مخزني**: تحويل بين المخازن
- **م扫描 الباركود**: كاميرا ماسح الباركود السريع
- **مجموعات الأصناف**: تصنيف الأصناف في مجموعات
- **التكلفة الوسيطية**: حساب التكلفة الوسيطية للأصناف
- **جرد المخازن**: جرد دوري مع المطابقة
- **صرف مخزني / صرف مخازني**: أنواع مختلفة من الصرف
- **توريد مخزني / توريد مخازني**: أنواع مختلفة من التوريد

### 2.5 نظام المال والصناديق
- **إضافة صندوق**: إضافة صناديق (صندوق/بنك)
- **حركات المال**: تحويل وحركات بين الصناديق
- **صرف عملة**: تحويل بين العملات
- **مصروفات وإيرادات**: تسجيل المصروفات والإيرادات
- **أموال العمالة**: عرض حركات الأموال لعمالة محددة
- **رواتب ومكافآت وعمالات**: إدارة الرواتب والمكافآت

### 2.6 نظام المستخدمين والمحمدين
- دعم عدة مستخدمين مع صلاحيات
- مراقبة بالكاميرا عند الدخول (أمن مشدد)
- تسجيل كامل لعمليات المستخدمين
- مشاركة البيانات مع المستخدمين (محلي/سحابي)
- نسخ احتياطي تلقائي محلي على Google Drive

## 3. نظام التقارير (المنافس)

### 3.1 تقارير الحسابات
| التقرير | الوصف |
|---------|-------|
| كشف حساب | ملخص الحركات لجميع الحسابات أو حساب محدد مع رصيد أول المدة |
| ميزان المراجعة | مجموع الحركات مدين ودائن لكل الحسابات |
| أرصدة الحسابات | عرض أرصدة الحسابات ذات الحركات فقط |
| حسابات بدون حركات | عرض الحسابات التي لم تجرى عليها عمليات |
| الحسابات الأصلية | عرض الحسابات التي ليس عليها عمليات (أرصدة فقط) |
| الحسابات الفرعية | عرض الحسابات التي عليها عمليات |

### 3.2 تقارير الفواتير
| التقرير | الوصف |
|---------|-------|
| ملخص الفواتير العامة | كشف ملخص بجميع فواتير العملاء مع الضرائب والخصم والحسم |
| ملخص الفواتير الآجلة | كشف ملخص بالفواتير الآجلة فقط |
| فواتير المبيعات النقدية | كشف بفواتير المبيعات النقدية |
| فواتير المبيعات الآجلة | كشف بفواتير المبيعات الآجلة |
| مرتجعات المبيعات | مرتجعات المبيعات النقدية والآجلة |
| مرتجعات المشتريات | مرتجعات المشتريات النقدية والآجلة |
| ملخص الفواتير المشتريات | فواتير المشتريات مع التفاصيل |
| ملخص الفواتير الآجلة مشتريات | فواتير المشتريات الآجلة فقط |
| إجمالي ربح الفواتير | حساب ربح الفواتير (الإجمالي والفرعي) |
| ربح الفاتورة | حساب ربح كل فاتورة على حدة |

### 3.3 تقارير المخزون
| التقرير | الوصف |
|---------|-------|
| ملف الأصناف | إدارة كاملة مع كميات وأسعار |
| حركات الأصناف | كشف تفصيلي لكل حركات الأصناف (وارد/صادر) |
| أصناف بدون حركات | الأصناف التي ليس عليها حركات |
| رصيد الأصناف | الرصيد الحالي لكل صنف |
| كشف تكلفة الأصناف | الكلفة لكل صنف |
| كشف حركات المخزون | ملخص حركات المخزون (بالصنف/المخزن) |
| كشف إجمالي الأصناف | الكمية الإجمالية حسب المخازن |
| جرد المخازن | نتيجة الجرد مع المطابقة |

### 3.4 تقارير المال والصناديق
| التقرير | الوصف |
|---------|-------|
| ملف المال | عرض حركات المال مع الأرصدة |
| حركات اليوم | كل حركات المال لليوم |
| حسب عمالة محددة | حركات مال لعمالة معينة |
| تحويلات الصناديق | تحويلات بين الصناديق |
| مصروفات وإيرادات | ملخص المصروفات والإيرادات |

### 3.5 تقارير السندات
| التقرير | الوصف |
|---------|-------|
| سندات القبض | كل سندات القبض |
| سندات الصرف | كل سندات الصرف |
| سندات التسوية | كل سندات التسوية |
| كشف السندات | عرض أرصدة السندات ومطابقاتها |
| سندات الجرد | نتيجة الجرد مع المطابقات |

### 3.6 تقارير الرسوم البيانية والإحصائيات

- رسم بياني للأرباح والخسائر

## 6. البنية التقنية
- **الإطار**: Xamarin.Android / .NET MAUI
- **قاعدة البيانات المحلية**: SQLite
- **قاعدة البيانات السحابية**: Firebase Realtime Database
- **المصادقة**: Firebase Authentication
- **الإشعارات**: Firebase Cloud Messaging
- **الخدمات السحابية**: Firebase (مزامنة، مصادقة، إشعارات)
- **النسخ الاحتياطي**: Google Drive
- **الشراء**: Google Play Billing v8.0
- **الإعلانات**: Google AdMob
- **ماسح الباركود**: ML Kit Barcode Scanning
- **الكاميرا**: CameraX
- **طباعة بلوتوث**: Bluetooth Printer
- **الخطوط المخصصة**: DUBAI-MEDIUM.TTF, GE_SS_TWO_MEDIUM.ttf, ariblk.ttf

## 7. نقاط القوة عند المنافس
1. شجرة حسابات هرمية متقدمة مع ترحيل سنوي

4. نظام مخازن متعدد مع تحويلات بين المخازن
5. طباعة بلوتوث مباشرة لطابعات 80mm
6. مشاركة التقارير كـ PDF وصورة وواتساب وSMS
7. نظام مستخدمين وصلاحيات متقدم
8. نسخ احتياطي سحابي عبر Google Drive
9. ماسح باركود مدمج (ML Kit)
10. عملات متعددة مع أسعار صرف
11. رسوم بيانية وإحصائيات
12. جرد المخازن مع المطابقة
